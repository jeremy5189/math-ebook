除法原理

給定兩個多項式 f( x ) 與 g( x ) ，且g( x )不是零多項式，用f( x ) 除以 g( x ) , 恰有一組多項式 q( x ) ,r( x ) 使

f( x ) ＝ g( x ) q( x ) ＋ r( x )
其中 r(x) ＝0 或 deg r( x ) ＞ deg g( x )

註：deg r( x )為該多項式的最高次方數

此時滿足上式的 q( x ) , r( x ) 分別稱為f( x )除以g( x )的商式,餘式

例題一

解答

餘式定理

f(x) ＝（x－a）q( x ) ＋ r
其中（x－a）是除式,故為求餘式需將除式設為 0 ,所以 x＝a 代入 即可得 r
故 r＝f(a) ， 此即是餘式定理。

延伸概念

若（x－a）為 f(x) 之因式，則 f(a) 之值等於 r(a) ，r 為一個函數，也可記為 r(x) ，次數依除式及被除式的次數而定。
