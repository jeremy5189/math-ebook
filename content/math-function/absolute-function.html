絕對值定義

在數線上一個點與原點的距離。其基本性質 :

  • 必不為負數
  • 0的絕對值為0
  • 互為相反數的兩數，其絕對值相等

運算性質

  • 1.|a| ≥ 0
  • 2.|ab| = |a||b|
  • 3.|a/b|= |a| / |b| (b≠0)
  • 4.|a+b|≤ |a| + |b|

絕對值函數

基本絕對值函數：y = f(x) = |x|

廣義定義：函數內含絕對值的函數，例：f(x) = log |x|

領導係數的影響

以下為 y = | ax | 的函數圖形，圖中紅色的小點可用手移動，上下移動可改變 a 值，請觀察 a 值(領導係數)改變後圖形的影響

將紅點往下拉可發現，a 趨近於零時的圖形接近一平行線。一但 a 小於 0 時，圖形會立刻回升，原因是因為絕對值會將負數變為正數。

結論

由於絕對值將負數轉為正數的特性，使得絕對值函數大多含有折線，意指圖形某一部分會有折疊往上的現象。
